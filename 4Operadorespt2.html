Relacionais

O resultado dessa expressão (Relacionais) sera sempre booleano, ou seja, verdadeiro ou falso!

  • > maior
  • < menor
  • >= maior ou igual
  • <= menor ou igual
  • == igual
  • != diferente
  • === identicos
  • !== desigual restrito
  • 5 > 2 -> True
    • O valor 5 é maior que o 2? sim, verdadeiro, e por ai vai
  • 7 > 4 -> false
  • 8 >= 8 -> True
  • 9 <= 7 -> false
  • 5 == 5 -> true
  • 4 != 4 -> false

Também podendo misturar os valores relacionais com os ARITMETICOS, mas sempre resolvendo os ARITMETICOS primeiro

ex: var a = 8
var b = 15
a(8) < b(15) = true
a(8) > b(15) = false
a(8) <= b(15) - 10 (Fazer primeiro o valor aritmetico que seria, b - 10, 15 - 10= 5, logo b é igual a 5.), = false, 8 é maior que 5.

Identidade

5 == 5 -> True
5 == '5' -> true, nesse caso ele não testa o valor da identidade e sim a grandeza.
5 === '5' -> false, nesse sinal ele indica igualdade, identicos, 5 não é identico a '5', o valor é o mesmo mas a identidade não.

Operadores Lógicos

que são eles:

  • ! -> Negação
    • ! é um operador unario, ou seja, so tem um operando isso é, ou é true ou é false
      ! True -> false (uma coisa que não é verdadeira é falsa)
      ! false -> tru1e (uma coisa que é falsa é verdadeira)
  • && -> Conjunção
    • && é um operador binario, isso é: dois operadores logicos
      true && true -> true
      true && false -> false
      false && true -> false
      false && false -> false
  • || -> Disjunção
    • Também outro operador binario que faz operações lógicas.
      basta que um seja verdadeiro para o resultado ser verdadeiro.
      true || True -> true
      true || False -> True
      false || True -> true
      false || false -> false
