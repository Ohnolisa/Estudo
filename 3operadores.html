ARITMETICOS e Atribuição

Quinta aula 
            
        ARITMETICOS: operadores 

        5 + 2 -> soma normal = 7 

        5 - 2 -> subtração normal = 3 

        5 * 2 -> multiplicação = 10 

        5 / 2 -> Divisão REAL, divisão de numeros reais, que leva a virgula = 2,5 

        5 % 2 -> Divisão INTEIRA, divisão que não leva numeros reais e virgulas = 1 

        5 ** 2 -> Potencia, significa: 5 ao quadrado = 25

A ordem de precedência entre os operadores 

        
        () -> Primeiro lugar os parenteses 

        ** -> depois dos parenteses as potencias 

        *, /, % -> Em seguida das potencias, temos a multiplicação, divisão e o resto da divisão 

        +, - -> e por fim as somas e subtrações.

Atribuição simples

Sempre cumprindo a ordem de precedências.

             
         Excemplos: 

         var a = 5 + 3: 

         variavel a recebe o valor de 5 mais 3 

         5 mais 3 é igual a 8. pegamos esse 8 e jogamos na var a

         

         var b = a % 5: 

         a variavel b recebe o valor de a(8) dividindo 5.
         a(8) dividido por 5 é 3, logo o valor de b é 3

         

         var c = 5 * b ** 2: 

         var c recebe o valor de 5 vezes b elevado a 2 

         o valor de b(3), 3 ao quadrado é 9, 9 vezes 5: 45

         

         var d = 10 - a / 2: 

         var d recebe 10 menos o valor de a(8) dividido por 2. 

         a(8) dividido por 2 é 4, 10 - 4 da 6

         

         var e = 6 * 2 /d: 

         var recebe 6 vezes 2 dividido por d(6). 

         6 vezes 2 é 12, 12 dividido por d(6) é 2

         

         var f = b % e + 4 / e: 

         var recebe b(3) dividido por e(2) mais 4 dividido por e(2). 

         b(3) dividido por e(2) é 1, 4 dividido por e(2) é 2. 1 + 2 é 3

Auto-atribuições

                
        var n = 3 

        variavel n recebe 3. Logo n é igual a 3. 

        

        n = n + 4 

        n recebe n mais 4. 

        Fazer a conta e depois jogar o valor como resultado do primeiro n. 

        n(3) mais 4 = 7. logo o valor de n passa a ser o 7. 

        

        n = n - 5 

        n recebe n menos 5 

        n(7) - 5 = 2. logo n passa a valer 2. 

        

        n = n * 4 

        n recebe n vezes 4 

        n(2) * 4 = 8. logo o n passa a valer 8 

        

        n = n / 2 

        n recebe n dividido por 2 

        n(8) / 2 = 4. logo o n passa a valer 4. 

        

        n = n **2 

        n recebe n elevado a 2 

        n(4) ** 2 = 16. logo n passa a valer 16 

        

        n = n % 5 

        n recebe n dividido por 5 

        n(16) % 5 = 1. logo n passa a valer 1.

Simplificando

            
        Escrever a variavel de forma resumida 
        Exemplo:
        n = n + 4. de forma resumida ficaria: n+=4. 

        Esse ex só serve se a mesma variavel receber ela mesma mais o numero 4. 

        

        n = n - 5. Ficaria n-=5 

        n = n * 4. Ficaria n*=4 

        n = n / 2. ficaria n/=2 

        n = n ** 2. Ficaria n**=2 

        n = n % 5. Ficaria n%=5
